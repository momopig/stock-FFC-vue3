# 用户管理

用户管理功能用于管理组织中的用户，支持添加、编辑、删除用户，支持设置用户角色，支持邀请用户注册。


## 添加成员

1. 点击左侧菜单中的“用户管理”，进入用户管理页面。
![用户管理](/images/userMgt/list.png)
2. 存在两种添加方式：
    - 通过邀请链接添加：点击邀请员工注册按钮，为邀请人选择角色，生成邀请链接后，将邀请链接发送给员工注册账号，即可自动将员工添加到组织中。
    ![邀请员工注册](/images/userMgt/invite-employee.png)
    - 通过直接新建员工账号：点击新建员工按钮，填写员工账号信息和角色，点击保存按钮，即可将员工添加到组织中。
    ![新建员工](/images/userMgt/new-employee.png)

::: tip 角色权限
系统目前内置三种角色：
- 超级管理员：拥有该公司的所有权限，可以管理该公司的所有员工和权限
- 店铺管理员：拥有活动管理、产品复制和仓库管理员的相关权限
- 仓库管理员：拥有SKC查询、发补货&查看物流的相关权限
:::
