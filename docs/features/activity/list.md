# 活动列表

活动列表功能用于查看和管理系统中的所有营销活动。

## 功能概述

活动列表页面提供了以下功能：
- 查看所有活动的基本信息
- 搜索和筛选活动
- 创建新活动
- 编辑现有活动
- 删除活动

## 页面布局

<!-- ![活动列表页面](/images/activity-list.png) -->

页面主要分为以下几个区域：

### 1. 搜索和筛选区域
- **活动名称搜索**：输入活动名称进行模糊搜索
- **状态筛选**：选择活动状态（进行中、已结束、未开始）
- **时间范围**：选择活动时间范围
- **查询按钮**：执行搜索操作

### 2. 操作按钮区域
- **新建活动**：创建新的营销活动
- **批量操作**：对选中的活动进行批量处理
- **导出数据**：将活动数据导出为 Excel 文件

### 3. 数据表格区域
显示活动的详细信息，包括：
- 活动名称
- 活动类型
- 开始时间
- 结束时间
- 状态
- 参与人数
- 操作按钮

## 操作步骤

### 查看活动列表

1. 登录系统后，点击左侧菜单"活动管理" → "活动列表"
2. 系统会自动加载所有活动数据
3. 使用滚动条查看完整列表

### 搜索活动

1. 在搜索框中输入活动名称的关键词
2. 选择相应的筛选条件
3. 点击"查询"按钮
4. 系统会显示符合条件的活动

::: tip 搜索技巧
- 支持模糊搜索，输入部分关键词即可
- 可以组合多个筛选条件
- 清空搜索条件可查看所有活动
:::

### 创建新活动

1. 点击"新建活动"按钮
2. 在弹出的表单中填写活动信息：
   - 活动名称（必填）
   - 活动类型
   - 开始时间
   - 结束时间
   - 活动描述
3. 点击"保存"按钮
4. 系统会创建新活动并刷新列表

### 编辑活动

1. 在活动列表中找到要编辑的活动
2. 点击该行右侧的"编辑"按钮
3. 在弹出的表单中修改活动信息
4. 点击"保存"按钮
5. 系统会更新活动信息

### 删除活动

::: danger 注意
删除活动是不可逆操作，请谨慎操作！
:::

1. 在活动列表中找到要删除的活动
2. 点击该行右侧的"删除"按钮
3. 系统会弹出确认对话框
4. 确认删除操作
5. 系统会删除活动并刷新列表

## 状态说明

| 状态 | 说明 |
|------|------|
| 未开始 | 活动尚未开始 |
| 进行中 | 活动正在进行中 |
| 已结束 | 活动已经结束 |
| 已暂停 | 活动被暂停 |

## 常见问题

### Q: 为什么看不到某些活动？
A: 可能的原因：
- 您没有查看该活动的权限
- 活动被筛选条件过滤掉了
- 活动数据加载中，请稍后刷新

### Q: 如何批量操作多个活动？
A: 操作步骤：
1. 勾选要操作的活动（使用表格左侧的复选框）
2. 选择批量操作类型
3. 确认操作

### Q: 活动数据可以导出吗？
A: 可以，点击"导出数据"按钮，系统会生成 Excel 文件供下载。
