# 活动编辑

活动编辑功能用于创建新的营销活动或修改现有活动的信息。

## 功能概述

活动编辑页面提供了以下功能：
- 创建新活动
- 编辑现有活动
- 设置活动基本信息
- 配置活动规则
- 预览活动效果

## 页面布局

<!-- ![活动编辑页面](/images/activity-edit.png) -->

页面主要分为以下几个区域：

### 1. 基本信息区域
- **活动名称**：输入活动的名称
- **活动类型**：选择活动类型（满减、折扣、赠品等）
- **活动描述**：详细描述活动内容
- **活动时间**：设置活动的开始和结束时间

### 2. 活动规则区域
- **参与条件**：设置用户参与活动的条件
- **优惠规则**：配置具体的优惠规则
- **使用限制**：设置使用次数和有效期限制

### 3. 商品选择区域
- **商品范围**：选择参与活动的商品
- **商品分类**：按分类选择商品
- **商品搜索**：搜索特定商品

### 4. 操作按钮区域
- **保存**：保存活动信息
- **预览**：预览活动效果
- **取消**：取消编辑操作

## 操作步骤

### 创建新活动

1. 在活动列表页面点击"新建活动"按钮
2. 系统会打开活动编辑页面
3. 填写活动基本信息：
   - 活动名称（必填）
   - 活动类型（必选）
   - 活动描述
   - 开始时间和结束时间
4. 配置活动规则：
   - 设置参与条件
   - 配置优惠规则
   - 设置使用限制
5. 选择参与活动的商品
6. 点击"保存"按钮
7. 系统会创建新活动并返回列表页面

### 编辑现有活动

1. 在活动列表中找到要编辑的活动
2. 点击该行的"编辑"按钮
3. 系统会打开活动编辑页面并加载现有数据
4. 修改需要更新的信息
5. 点击"保存"按钮
6. 系统会更新活动信息

### 预览活动效果

1. 在编辑页面填写完基本信息后
2. 点击"预览"按钮
3. 系统会显示活动的前端效果
4. 确认效果无误后继续编辑

::: tip 编辑建议
- 活动名称要简洁明了，便于用户理解
- 活动时间要合理设置，避免时间冲突
- 优惠规则要清晰明确，避免歧义
- 建议先预览效果再保存
:::

## 字段说明

### 基本信息字段

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 活动名称 | 活动的显示名称 | 是 |
| 活动类型 | 活动的类型分类 | 是 |
| 活动描述 | 详细的活动说明 | 否 |
| 开始时间 | 活动开始的时间 | 是 |
| 结束时间 | 活动结束的时间 | 是 |

### 活动规则字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 参与条件 | 用户参与活动的条件 | 满100元 |
| 优惠规则 | 具体的优惠方式 | 减20元 |
| 使用限制 | 使用次数和有效期 | 每人限用1次，7天内有效 |

## 常见问题

### Q: 活动时间可以修改吗？
A: 可以，但需要注意：
- 如果活动已经开始，修改时间可能影响已参与的用户
- 建议在活动开始前完成时间设置

### Q: 如何设置复杂的优惠规则？
A: 系统支持多种优惠规则组合：
- 满减：满X元减Y元
- 折扣：X折优惠
- 赠品：满X元送Y商品
- 包邮：满X元免运费

### Q: 活动保存后可以立即生效吗？
A: 活动保存后需要审核通过才能生效，具体流程请咨询管理员。
