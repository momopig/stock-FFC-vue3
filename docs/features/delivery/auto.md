# auto自动发货配置

auto自动发货配置功能用于配置自动发货规则，支持自动发货、发送下载箱唛。

## 自动发货流程

### 1. 配置自动发货规则

- 点击新建自动发货配置项（SKC）栏目
- 点击新建SKC
- 输入要自动发货的的SKC号，会自动填充店铺和SKC名称（需要事先录入店铺信息）
- 选择发货地址，如果该发货地址未配置供应商，则点击弹出的“现在录入”按钮，打开弹窗，进行[供应商信息录入](/features/supplier/list)
- 填写SKC箱规
- SKU不配置则默认与SKC一致，可以单独针对某个SKU设置发货地址、箱规

### 2. 自动发货

- 点击自动发货栏目
- 选择需要执行店铺、输入需要执行自动发货的的SKC或者备货单号（需要事先配置号对应的SKC自动发货配置）
- 选择需要执行自动发货到哪一个步骤
  - 仅到待装箱发货：仅将发货单添加到待装箱发货
  - 今日发货：自动发货，设置发货日期为今日下午7点
  - 明日发货：自动发货，设置发货日期为明日下午7点
- 自动发货会按照SKC对应配置的发货地址、箱规进行发货。物流会优先选择SKC对应的供应商配置的非黑名单物流，且是价格最低的物流。自动发货的时间点默认为下午7点
