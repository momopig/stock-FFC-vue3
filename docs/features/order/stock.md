# 备货单查询

备货单查询功能用于查看和管理商品的备货情况，确保库存充足。

## 功能概述

备货单查询页面提供了以下功能：
- 查看备货单列表
- 搜索特定备货单
- 查看备货单详情
- 更新备货状态
- 导出备货数据

## 页面布局

<!-- ![备货单查询页面](/images/order-stock.png) -->

页面主要分为以下几个区域：

### 1. 搜索筛选区域
- **备货单号**：输入备货单号进行搜索
- **商品名称**：按商品名称筛选
- **备货状态**：选择备货状态
- **时间范围**：选择备货时间范围

### 2. 操作按钮区域
- **新建备货单**：创建新的备货单
- **批量操作**：批量处理备货单
- **导出数据**：导出备货单数据

### 3. 备货单列表区域
显示备货单的详细信息，包括：
- 备货单号
- 商品信息
- 备货数量
- 备货状态
- 创建时间
- 操作按钮

## 操作步骤

### 查看备货单列表

1. 登录系统后，点击左侧菜单"订单管理" → "备货单查询"
2. 系统会自动加载所有备货单
3. 使用搜索和筛选功能查看特定备货单

### 搜索备货单

1. 在搜索框中输入备货单号或商品名称
2. 选择相应的筛选条件
3. 点击"查询"按钮
4. 系统会显示符合条件的备货单

### 查看备货单详情

1. 在备货单列表中找到要查看的备货单
2. 点击该行的"详情"按钮
3. 系统会显示备货单的详细信息：
   - 商品详细信息
   - 备货数量
   - 备货状态
   - 操作历史

### 更新备货状态

1. 在备货单列表中找到要更新的备货单
2. 点击该行的"更新状态"按钮
3. 在弹出的对话框中选择新的状态
4. 添加备注信息（可选）
5. 点击"确认"按钮
6. 系统会更新备货单状态

## 状态说明

| 状态 | 说明 |
|------|------|
| 待备货 | 备货单已创建，等待备货 |
| 备货中 | 正在准备商品 |
| 已备货 | 商品已准备完成 |
| 已发货 | 商品已发出 |
| 已完成 | 备货流程已完成 |

## 常见问题

### Q: 如何创建新的备货单？
A: 创建备货单的步骤：
1. 点击"新建备货单"按钮
2. 选择要备货的商品
3. 设置备货数量
4. 选择备货仓库
5. 保存备货单

### Q: 备货单可以修改吗？
A: 只有"待备货"状态的备货单可以修改，其他状态的备货单只能更新状态。

### Q: 如何批量处理备货单？
A: 批量处理步骤：
1. 勾选要处理的备货单
2. 选择批量操作类型
3. 确认批量操作
