# 发货管理

发货管理功能用于处理订单的发货流程，确保商品能够及时送达客户手中。

## 功能概述

发货管理页面提供了以下功能：
- 查看待发货订单
- 批量处理发货
- 打印发货单
- 跟踪物流信息
- 处理异常订单

## 页面布局

<!-- ![发货管理页面](/images/order-deliver.png) -->

页面主要分为以下几个区域：

### 1. 订单筛选区域
- **订单状态**：筛选不同状态的订单
- **时间范围**：选择订单时间范围
- **商品类型**：按商品类型筛选
- **查询按钮**：执行筛选操作

### 2. 批量操作区域
- **批量发货**：选择多个订单进行批量发货
- **批量打印**：批量打印发货单
- **导出数据**：导出订单数据

### 3. 订单列表区域
显示订单的详细信息，包括：
- 订单编号
- 客户信息
- 商品信息
- 订单金额
- 订单状态
- 操作按钮

## 操作步骤

### 查看待发货订单

1. 登录系统后，点击左侧菜单"订单管理" → "发货管理"
2. 系统会自动加载待发货的订单
3. 使用筛选条件查看特定订单

### 单个订单发货

1. 在订单列表中找到要发货的订单
2. 点击该行的"发货"按钮
3. 在弹出的对话框中确认发货信息：
   - 物流公司
   - 运单号
   - 发货备注
4. 点击"确认发货"按钮
5. 系统会更新订单状态为"已发货"

### 批量发货

1. 勾选要发货的订单（使用表格左侧的复选框）
2. 点击"批量发货"按钮
3. 在弹出的对话框中设置发货信息
4. 确认批量发货操作
5. 系统会批量更新订单状态

::: tip 发货建议
- 发货前请仔细核对订单信息
- 确保商品包装完好
- 及时更新物流信息
- 保留发货凭证
:::

## 状态说明

| 状态 | 说明 |
|------|------|
| 待发货 | 订单已确认，等待发货 |
| 已发货 | 商品已发出，等待配送 |
| 配送中 | 商品正在配送途中 |
| 已送达 | 商品已送达客户 |
| 异常 | 配送过程中出现异常 |

## 常见问题

### Q: 如何处理异常订单？
A: 异常订单处理流程：
1. 在订单列表中找到异常订单
2. 点击"处理异常"按钮
3. 选择异常类型和处理方案
4. 提交处理结果

### Q: 可以修改已发货的订单吗？
A: 已发货的订单只能修改物流信息，不能修改商品信息。

### Q: 如何跟踪物流信息？
A: 系统会自动同步物流信息，您也可以手动更新运单号。
