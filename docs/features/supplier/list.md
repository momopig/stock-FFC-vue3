# 供应商列表

供应商配置功能用于将供应商和对应的发货地址进行关联。支持设置发货黑名单，以及通过[WebHook key](/features/monitor/webhook)将供应商信息发送到企微群里

## 功能概述

- 配置供应商和对应的发货地址，支持添加、编辑、删除
- 支持设置发货黑名单（即该供应商对应的SKC，自动发货时的物流不会选择黑名单里的物流）
- 支持设置[WebHook key](/features/monitor/webhook)将供应商信息发送到企微群里（自动发货时，将箱唛、物流单号等信息发送到企微群里）
- 支持复制供应商关联的SKC（需要在[自动发货配置](/features/delivery/auto)中配置SKC的供应商）
