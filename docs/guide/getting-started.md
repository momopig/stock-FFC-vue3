# 快速开始

本指南将帮助您快速了解商城系统的基本操作流程。

## 注册账号

1. 从登录页面点击注册按钮，进入注册页面
![注册账号](/images/register/register-entry.png)
2. 填入注册信息，点击注册。当前注册账号为该公司组织的主账号，主账号拥有该公司的所有权限。
![注册账号](/images/register/register-page.png)
3. 使用注册的账号和密码登录系统，登录成功后，会进入系统首页。
![系统首页](/images/home/home-page.png)

## 添加成员

1. 点击左侧菜单中的“用户管理”，进入用户管理页面。
![用户管理](/images/userMgt/list.png)
2. 存在两种添加方式：
    - 通过邀请链接添加：点击邀请员工注册按钮，为邀请人选择角色，生成邀请链接后，将邀请链接发送给员工注册账号，即可自动将员工添加到组织中。
    ![邀请员工注册](/images/userMgt/invite-employee.png)
    - 通过直接新建员工账号：点击新建员工按钮，填写员工账号信息和角色，点击保存按钮，即可将员工添加到组织中。
    ![新建员工](/images/userMgt/new-employee.png)

::: tip 角色权限
系统目前内置三种角色：
- 超级管理员：拥有该公司的所有权限，可以管理该公司的所有员工和权限
- 店铺管理员：拥有活动管理、产品复制和仓库管理员的相关权限
- 仓库管理员：拥有SKC查询、发补货&查看物流的相关权限
:::

## 录入店铺信息
1. 点击左侧菜单中的“店铺管理”，进入店铺管理页面。
2. 点击“新建店铺”按钮，进入店铺新建页面。
3. 填写店铺信息，点击“保存”按钮，即可将店铺信息录入系统。
![店铺信息](/images/mall/record-mall.png)

## 主要功能区域

### 顶部导航栏
- **标签页**：显示当前打开的功能页面，支持多页签
- **添加按钮**：快速打开新功能
- **用户信息**：显示当前登录用户

### 左侧菜单
- **功能分类**：按业务模块组织功能
- **权限控制**：只显示您有权限访问的功能

### 主内容区
- **功能页面**：具体的业务操作界面
- **数据展示**：表格、图表等数据呈现


## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + left` | 返回上一个页签 |
| `Alt + right` | 前进下一个页签 |
| `Ctrl + F` | 页面内搜索 |
| `F5` | 刷新页面 |
| `Esc` | 关闭弹窗 |
