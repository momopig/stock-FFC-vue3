## 风险信号
股票列表接口的quote字段添加了风险信号数据，risk_signs字段。现在需要在列表加一列，风险信号
```json
risk_signs: [
  {
    "sign": "股价跌破3日均线",
    "explain": "主升浪的强势上涨线，跌破表示强势有减弱迹象，可提前止盈，减少回撤"
  }
]
```
1、如果risk_signs数组项为空，则展示”无“
2、如果risk_signs数组项不为空，则展示risk_signs数组项的sign字段，作为标签展示。explain字段作为标签的提示文字，hover展示。


## 搜索
1、搜索改为调用接口来搜索，不再使用前端搜索
2、查询接口参数支持
- exchange_code (query)：交易所代码（SH/SZ/HK/US）
- strategy_name (query)：策略名称模糊查询
- stock_code (query)：股票代码模糊查询
- stock_name (query)：股票名称模糊查询
